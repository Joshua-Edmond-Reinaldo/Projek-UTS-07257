ZenTask: Aplikasi To-Do List Harian

Proyek Sederhana: Frontend Saja

ZenTask adalah aplikasi daftar tugas (To-Do List) berbasis browser yang dirancang untuk membantu pengguna mengelola tugas harian mereka dengan antarmuka yang bersih dan minimalis. Fokus utama proyek ini adalah manipulasi DOM (Document Object Model) menggunakan JavaScript murni dan penyimpanan data pada Local Storage browser sehingga data tugas tetap tersimpan meskipun halaman di-refresh.

  • Penambahan dan penghapusan tugas baru secara dinamis.
  • Menandai tugas sebagai selesai (checked) dengan efek visual.
  • Penyimpanan data otomatis di Local Storage.
  • Antarmuka yang responsif di berbagai ukuran layar.
